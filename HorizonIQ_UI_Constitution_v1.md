# HorizonIQ UI Constitution

## Design Philosophy

HorizonIQ should feel:

* Intelligent
* Premium
* Trustworthy
* Modern
* Clear

Not futuristic for the sake of being futuristic.

## Layout Principles

* Large whitespace
* Clear information hierarchy
* Responsive by default
* Mobile-first

---

## Dashboard Principles

The dashboard must answer:

1. What is changing?
2. Why does it matter?
3. What should I do?

Every section must support one of these questions.

---

## Animation Principles

Motion should:

* Guide attention
* Explain relationships
* Improve comprehension

Never animate purely for decoration.

---

## Success Criteria

A first-time visitor should understand:

* What HorizonIQ does
* Why it is useful
* What action to take

within 10 seconds.
